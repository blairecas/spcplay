MS-0511 (UKNC) version of Manwe's SPCPlay (BK-0010/11)
Original: https://manwe.pdp-11.ru/?/music/soft

Playing .SPC+.INS files on covox device. Both LPT port A and AY reg #14 are supported.

Usage: RUN SPCP FILE[/Y] (both FILE.SPC and FILE.INS must be present on device)
       /Y - will try to play on AY reg #14 instead of default LPT port A
